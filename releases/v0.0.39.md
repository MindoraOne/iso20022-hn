## 🙏 Why This Release

After releasing v0.0.38 to PyPI with comprehensive documentation improvements, we discovered the docs badge in README.md was pointing to an incorrect URL. Since PyPI versions are immutable, we're releasing v0.0.39 with the corrected URL to ensure users see the badge linking to the right documentation site.
